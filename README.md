# OCR

### Important infromation
You need to add ***C:\OCR_Path\leptonica\lib\*** to the system path to be able to run the projects.

Directory structure needs to look like this:
<pre>
FANN
images
leptonica
    bin
    include
    leptonica-1.XX
    lib
<b>Projects</b>
</pre>
<br />
